Projects
I have many projects ranging from software I've programmed to games. This section only includes the computer science based projects I've worked on.
Business Intelligence

I was tasked, along with my group of 3 other people, by MAXX Power to gather user data on their website. My group chose to add Google Analytics in conjunction with Google Data Studio to the site, and from there we created a dashboard to present, executive summary, and PowerPoint on the data collected.

Information Security

My group was given the goal by MAXX Power to find a solution to the company's problem with password management. We made a list of criteria the password manager would have to meet, made a list of possible password managers, created an implementation strategy, wrote an executive summery, and put together a presentation on our findings.

Development

My group was requested by MAXX Power to redesign their site. To do this we first obtained a copy of the original source code, designed a new site layout, included new features, made any important fixes that were required, and designed a quality control checklist to ensure the site worked well. We then wrote up an executive summery.

History SOL simulator

I made a Virginia & United States History SOL simulator with a group of 2 other people as the final exam for the duel enrollment Java course I took. It gives the user 60 released sol questions to answer as practice for the real SOL. It will then gather user responses, calculate the users grade, and tell the user weather or not they passed.

Forest Escape

Forest Escape is a text-bases RPG game I made by myself using python turtle. It follows the story of the main character and their friend getting lost in a forest. The player then has to find a way to get out using multiple methods. It is still a very early work in progress and not finished yet.
